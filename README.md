## Getting Started

To get a local copy up and running follow these simple steps:

---

### Prerequisites

* npm
```sh
npm install npm@latest -g
```

---

### Installation

1. Clone the repo
```sh
git clone https://github.com/JakeAdmire/Rotten-Potatoes
```
2. Install NPM packages
```sh
cd Rotten-Potatoes && npm install
```
